## Admin Roles

1. **SUPER_ADMIN** - Full system access (all pages, all actions)
2. **FINANCE_ADMIN** - Money only (deposits, withdrawals, balances)
3. **OPERATIONS_ADMIN** - Games only (bets, casino, virtual)
4. **SUPPORT_ADMIN** - Read-only (view only, no actions)

## Betting Rules

- **Max Selections** - 60 per bet slip
- **Bonus** - 3% on winnings
- **Max Payout** - 200,000,000 NGN (silent cap)
- **Withdrawal Requirement** - 2 settled bets (or 1 if promo code used)
